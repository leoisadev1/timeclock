Convex local setup
------------------

The local Convex deployment stores state under:

packages/backend/.convex/local/default/

That directory is ignored by git and contains the local SQLite-backed Convex
state for this checkout.

To initialize or refresh the local deployment without logging in to Convex cloud:

cd packages/backend
CONVEX_AGENT_MODE=anonymous node_modules/.bin/convex dev --once --local-cloud-port 3210 --local-site-port 3211 --typecheck disable

If auth config fails because CLERK_JWT_ISSUER_DOMAIN is missing from the local
deployment, set it on the local deployment:

cd packages/backend
CONVEX_AGENT_MODE=anonymous node_modules/.bin/convex env set CLERK_JWT_ISSUER_DOMAIN https://modern-kite-48.clerk.accounts.dev
